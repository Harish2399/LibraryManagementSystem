# 📚 Library Management System

A desktop-based Library Management System developed using Python, Tkinter, and SQLite.

## Features

- Login
- Add Book
- View Books
- Search Book
- Delete Book
- Issue Book
- Return Book
- Export Books to CSV
- Dashboard Statistics

## Technologies Used

- Python
- Tkinter
- SQLite

## How to Run

```bash
python login.py
```
